# Project: NeuroTracker AI — Цифровой помощник дистрибьютора CoralClub
> Последнее обновление: март 2026

---

## Что это за проект

NeuroTracker AI — цифровой инструмент для дистрибьюторов CoralClub, который помогает расшифровывать диаграммы Трекера Здоровья и превращать их в готовые тексты для клиентов. Инструмент закрывает разрыв между «знаю теорию» и «клиент купил программу» — даёт конкретные шаги на каждом этапе работы с клиентом, от первого поста до завершения программы восстановления. В основе — методология врача-практика Ольги Алексеевны Бутаковой (40+ лет опыта).

---

## Для кого

**Основная аудитория:** дистрибьютор CoralClub с Трекером Здоровья (опыт 0.5–5 лет), который знает теорию, но не получает практического результата — клиенты интересуются, но не покупают программы.

**Дополнительная аудитория:** лидер-наставник с командой (5+ лет), который тратит часы на обучение каждого новичка работе с диаграммами.

**Автор и создатель:** Валерий Созанов — дистрибьютор CoralClub, Россия.

## Стиль дизайна

### Два стиля в проекте (разные задачи)

**NeuroTracker AI (index.html, landing.html) — технологичный тёмный:**
- Фон: `#080817` — глубокий тёмно-синий
- Акценты: фиолетовый `#7C3AED` + cyan `#06B6D4`
- Карточки: glassmorphism (`rgba(255,255,255,0.05)` + `backdrop-filter: blur`)
- Шрифт: системный стек, крупные заголовки (`clamp(40px, 7vw, 80px)`)
- Настроение: экспертность, технологичность, доверие без заносчивости
- Анимации: scroll-driven fade-in, marquee, счётчики, hover-glow

**Визитка (vizitka.html) — постерный тёмно-янтарный:**
- Фон: `#060402` — почти чёрный с тёплым оттенком
- Акценты: золото `#f0a500` + оранж `#e85d04`
- Фото: занимает всю карточку как фон (poster/editorial style)
- Градиент снизу растворяет фото в тёмную зону с текстом
- Настроение: личность, живое присутствие, тепло

### Общие принципы для обоих стилей
- Один HTML-файл, без внешних зависимостей, без CDN
- Mobile-first (70%+ аудитории — телефон)
- Все иконки — SVG, фото — base64 внутри файла
- Нет MLM-риторики («прорыв», «трансформация», «уникальный»)
- Дисклеймер «не является медицинским диагнозом» — на видном месте
